Создайте в классе ApplicationContext следующий метод:

	public <T extends Object> T getBean(Class c) {
		return (T) beans.get(c);
	}

Идея этого метода заключается в том, чтобы дать возможность
получать созданную при стартапе приложения инстанцию любого
класса хранящегося в мапе beans. Этот метод не просто достаёт
из мапы инстанцию объекта но и приводит ее к нужному типу данных.
