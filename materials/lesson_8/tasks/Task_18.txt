Бизнес сценарий: вы хотите написать письмо читателю с напоминанием
о не сданных в библиотеку книгах. Напомним, что признак не сданной
книги является отсутствие данных в поле book_return_date таблицы
reader_books. Вам известно только id читателя которому вы хотите
написать письмо с напоминанием.

Составьте SQL запрос который по id читателя вернёт из базы все
не сданные книги (название и автора).

При написании команды воспользуйтесь SQL командой join.

Написанную команду сохраните в файле dbJoinScript.sql.
