Бизнес сценарий: вы хотите ответить на вопрос кто из читателей
брал на чтение книгу? Вам известно только id книги.

Составьте SQL запрос который по id книги вернёт из базы всех
читателей (имя и фамилия) бравших на чтение данную книгу.

При написании команды воспользуйтесь SQL командой join.

Написанную команду сохраните в файле dbJoinScript.sql.
