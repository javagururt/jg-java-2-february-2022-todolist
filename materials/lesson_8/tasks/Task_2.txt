Создайте в корне вашего проекта (на уровне с папкой src)
папку под названием database.
В этой папке мы будем хранить SQL скрипты создания базы данных.
В папке database создайте два файла:
 - dbCreationScript.sql  - это скрипт для создания структуры базы данных;
 - dbDataInsertScript.sql - это скрипт для заполнения базы данных
   тестовыми данными.
