После того как структура базы данных создана её можно заполнить данными.

Напишите следующие варианты команды SQL insert для таблиц
books/readers/reader_books:
    - команда insert с указанием только обязательных полей и без
      явного указания значения для поля id (так чтобы значение
      для поля id сгенерила сама база данных при помощи стратегии
      AUTO_INCREMENT);
    - команда insert с указанием только обязательных полей и
      явным указанием значения для поля id;
    - команда insert с указанием всех полей в таблице.

Составленные вами SQL команды проверьте на реальной базе данных
выполнив их.

Написанные команды сохраните в файле dbDataInsertScript.sql.
