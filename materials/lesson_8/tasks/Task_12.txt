После того как вы заполнили базу данных тестовыми данными
при помощи SQL команды insert пришло время поэксперементировать
с SQL командой select.

Создайте для таблиц books/readers/reader_books различные
варианты команды select:
  - выборка всех данных из таблицы;
  - выборка данных по id;
  - выборка данных по одному свойству (не id);
  - выборка данных по нескольким свойствам используя AND и OR;
  - выборка с сортировкой по указанному полю;
  - выборка с указанием лимита возвращаемых строк;
  - ваши варианты SQL команды select.

Составленные вами SQL команды проверьте на реальной базе данных
выполнив их.

Написанные команды сохраните в файле dbDataSelectScript.sql.
