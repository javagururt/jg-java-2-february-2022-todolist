Создайте в пакете dependency_injection класс DIComponentCreator.
В классе DIComponentCreator создайте следующий метод:

    public void create(ApplicationContext applicationContext,
    				   List<Class> diComponents)

Цель этого метода создать инстанции всех классов переданных
в качестве параметра List<Class> diComponents. Для создания
инстанций (экземпляров) классов в момент выполнения программы
можно например воспользоваться конструктором по умолчанию и
механизмом reflection.

Созданные при помощи конструктора по умолчанию инстанции
нужно добавить в аппликационный контекст при помощи вызова
метода:
   applicationContext.addBean(Class beanClass, Object beanInstance)

Попробуйте самостоятельно реализовать указанный выше метод.

то переходите к следующему заданию в котором будет приведено
одно из возможных решений.
