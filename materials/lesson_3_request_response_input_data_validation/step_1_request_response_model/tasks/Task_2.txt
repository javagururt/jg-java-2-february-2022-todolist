Создайте класс AddXXXRequest и добавьте в него все необходимые параметры
введённые пользователем для создания объекта. Например название и автора книги,
или название и описание продукта. Эти параметры должны передаваться в
класс запроса через конструктор и сохраняться в соответствующих
свойствах класса. Так же создайте get методы. Set методы создавать не надо.
