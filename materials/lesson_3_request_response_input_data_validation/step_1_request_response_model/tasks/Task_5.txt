Создайте класс RemoveXXXRequest и добавьте в него все необходимые параметры
введённые пользователем для удаления объекта. Например id объекта.
Эти параметры должны передаваться в
класс запроса через конструктор и сохраняться в соответствующих
свойствах класса. Так же создайте get методы. Set методы создавать не надо.
