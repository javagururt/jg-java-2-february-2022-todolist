Создайте класс RemoveXXXResponse и добавьте в него признак
был ли удалён объект или нет (boolean параметр).
Признак был ли удалён объект должен передаваться в
класс ответа через конструктор и сохраняться в соответствующем
свойстве класса. Так же создайте get метод для получения этого признака.
Set метод создавать не надо.
