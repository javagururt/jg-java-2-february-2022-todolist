Выбор лидера команды.

"Нет плохих команд, есть только плохие лидеры".



Для успеха команды нужно выбрать лидера.



Лидеры бывают двух типов:
- технический, тот кто больше всех знает
  или имеет больше всего практического опыта;
- управленческий, тот кто умеет управлять людьми
  и процессами производства.

Вы можете выбрать одного или двух лидеров.
Один может быть технический, другой управленец.
Уникальный случай если это будет один и тот же человек.

Для выбора лидера проведите в своей команде stand up.

Роль лидера может в будущем передаваться другим членам команды.
