Параметры класса SearchBooksRequest:
    private Integer pageNumber;
    private Integer pageSize;
являются не обязательными, пользователь их может указывать в запросе
или не указывать. Правила валидации для этих параметров:
- они должны быть не указаны или указаны оба вместе;
- pageNumber должен быть больше 0;
- pageSize должен быть больше 0.

Реализуйте правила валидации для этих параметров на стороне
бизнес логики.
