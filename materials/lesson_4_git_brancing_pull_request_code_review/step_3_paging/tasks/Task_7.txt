Функциональность постраничного просмотра данных является
независимой от поиска книг и может быть применена так же в другом месте.
Например при поиске читателей. Следуя принципу SRP нужно
реализовать функционал постраничного просмотра данных
по возможности отдельно.

В запросе SearchBooksRequest параметры постраничного
просмотра данных указываются при помощи двух свойств:

    private Integer pageNumber;
    private Integer pageSize;

Создайте класс Paging и перенесите в него свойства
касающиеся постраничного просмотра данных.
Класс Paging создайте в пакете /core/requests.
