Реализуйте на стороне бизнес логики функционал
отбора запрошенной пользователем страницы данных
при поиске книг в SearchBooksService.
Если это необходимо, то внесите изменения в методы
базы данных.
Помните о том, что функционал отбора запрошенной
страницы является не обязательным. Если пользователь
не указал соответствующие параметры в запросе, то
ему возвращаются все найденные данные.
