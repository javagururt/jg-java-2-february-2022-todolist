Поиск книг можно реализовать двумя способами:
1. Получать все книги у базы данных и потом в сервисе
   выполнять поиск;
2. Осуществлять поиск в базе данных напрямую, так чтобы
   она возвращала уже нужный список.

Первый вариант не правильный и не будет работать при больших
объёмах данных в юазе данных. Представьте себе, что в базе данных
хранятся миллионы записей. Вы будете доставать все эти записи
из базы данных в приложение и потом только осуществлять поиск?

Поиск нужно осущетвлять напрямую в базе данных.
