Язык SQl (Structured Query Language) является универсальным
языком общения с реляционными базами данных. С помощью этого
языка можно сформировать любую команду к СУБД (Системе
Управления Базами Данных).

Для изучения основ SQL воспользуйтесь онлайн туториалом:

https://sqlbolt.com/lesson/select_queries_introduction
