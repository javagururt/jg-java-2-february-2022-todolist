Убедитесь что в доменных классах есть дефолтные конструкторы
(конструкторы без параметров). Без них ORM мапинг работать не
будет.

public class Book {

	public Book() { }

	public Book(String title, String author) {
		this.title = title;
		this.author = author;
	}

}

По мимо дефолтного конструктора в классе доменного объекта могут
находиться и другие конструкторы. Для ORM важно наличие именно
дефолтного конструктора.
